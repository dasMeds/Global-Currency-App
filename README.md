# Global-Currency-App

Global Currency Archive
A simple project that fulfilled the requirements of our Data Structure and Algorithm course. This utilizes java and java swing.

Features
Extensive Currency Database: Contains data for over 180 world currencies across multiple regions including Asia, Europe, North America, South America, Africa, Middle East, Oceania, and the Caribbean.

Interactive Search & Filtering: Search by currency code or name, and filter results using region-specific toggles.

Denomination Visualizer: Dynamically generates custom UI icons to represent specific coins and banknotes based on their monetary value and threshold.

Lore & Analysis Generator: Programmatically creates a 10-point historical and economic summary for selected currencies, detailing the central bank, material composition, and future outlook.

Currency Conversion Estimates: Provides real-time estimated conversions into USD, EUR, GBP, and PHP based on base rates.

Export Functionality: Allows users to export the full currency report and conversion data to a local text file.

Custom Dark UI: Features a custom dark theme with customized scrollbars, buttons, and responsive hover effects.

Application Flow
Select Currency: Use the search bar or region filters to find a specific global currency on the main page.

Treasury: View the available denominations for the selected currency and choose a specific coin or bill.

Analysis: Read the generated historical lore, view the currency specifications, and see the conversion value. Click the export button to save the text report to your local machine.
